# 📱 Campsite Commander

## App Description:
The Campsite Commander app is an Android application developed utilising Kotlin through Android Studio. The intetion of the app is to assist users to pack before 
departing for an outdoor adventure, example: a camping trip. 

## Features Include:
- App displays 3 screens (Splash Screen, Main screen, Detailed View screen).
- User enters the Splash screen and is automatically taken to the Main screen in 3 seconds. 
- 10 essentials loaded onto the app. 
- User can add their own item of choice. 
- Add gear button.
- Load essentials button.
- Detailed view screen displays all items the user has added. 
- Back to base button.
- Back to base button takes user back to the splash screen.

 ## Built with:
- Kotlin
- Android Studio
- GitHub (version control)

### Installation:
1. Clone the repo
2. Open in Android Studio
3. Run the app
